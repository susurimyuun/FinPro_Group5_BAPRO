# FinPro_Group5_BAPRO
Topic : SDG 17 (Strengthen the means of implementation and revitalize the Global Partnership for Sustainable Development)

Concept : A tinder like program that can compare between users country and other countries in Asia to find the perfect partner through different methodes and 
          SDGS-Focus.
The program can then compare in certain radius of categories given to the user and the databases of Asian Countries.

After then the program will suggests up to 3 best candidates for negotiate after with the user.

This program also mainly focuses on collaborating with Asian countries.
